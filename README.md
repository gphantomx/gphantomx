## 👨‍💻 About Me

I'm a developer passionate about building scalable full-stack and realtime applications. I’ve worked on projects ranging from frontend clones and realtime systems to Web3 applications and smart contracts on Solana and Ethereum.

My experience includes building chat applications, multiplayer realtime systems, crypto payment solutions, staking platforms, and competitive programming platforms while exploring system design, distributed systems, and DevOps fundamentals.

Beyond development, I actively explore competitive programming, cybersecurity, and research-oriented fields like quantum computing, with a strong focus on learning through building.
